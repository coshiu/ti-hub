STAGE 01
Intake & Role Understanding
📋
PRIMARY GOAL
Get technical clarity fast so you can calibrate with confidence.
🤖 How to use AI
📚
Blueprint Creation: Summarize intake notes into a role blueprint covering scope, level, must-haves, and differentiators.
🧩
Skill Decomposition: Break down key skill clusters (e.g., "What are core subdomains of storage engineering?").
🌐
Market Alignment: Compare internal descriptions vs. market equivalents to align terminology.
⟩_ Example Prompt
⚙ Prompt
"Summarize this job description into 3 buckets: core systems knowledge, domain-specific expertise, and differentiators between senior and staff-level talent."
📈 Expected Results
✓ Shared language with hiring manager before search starts
✓ Clear must-haves vs. nice-to-haves definition
✓ Faster, higher-confidence calibration cycles
STAGE 02
Market Mapping
🗺️
PRIMARY GOAL
Identify where qualified talent actually sits — faster and cleaner.
🤖 How to use AI
📋
Target Generation: Generate lists of target companies by tech stack, product type, or infrastructure model.
🏢
Org Classification: Use AI to classify orgs (e.g., "Which companies have built distributed file systems?").
🔭
Signal Detection: Cross-check public profiles for emerging players or acquisition spin-offs.
⟩_ Example Prompt
⚙ Prompt
"List companies known for developing hypervisor or kernel-level technology in the AI/Cloud Infra space."
📈 Expected Results
✓ Strategic market maps instead of scattershot sourcing
✓ Better targeting and higher hit rates
✓ Discovery of hidden talent pools
STAGE 03
Search String Design
🔍
PRIMARY GOAL
Build smarter boolean strings and avoid keyword bias in your sourcing.
🤖 How to use AI
⚙
Generate Variations: Create complex boolean variations for niche roles automatically.
🔤
Synonym Detection: Spot synonyms, alternate titles, and jargon you might miss.
✏️
Pre-Testing: Test strings for potential false positives before running them in LinkedIn or SeekOut.
⟩_ Example Prompt
⚙ Prompt
"Create a boolean string for senior engineers experienced in hypervisor or virtualization stack development (KVM, Xen, QEMU, VMware). Exclude front-end and QA roles."
📈 Expected Results
✓ Cleaner pipelines with reduced noise
✓ Higher signal-to-noise ratio in search results
✓ Broader coverage of relevant titles and skills
STAGE 04
Calibration & Pattern Recognition
⚖️
PRIMARY GOAL
Strengthen judgment through comparison, not volume.
🤖 How to use AI
👥
Profile Comparison: Compare two profiles side-by-side and ask AI to summarize relative strengths or potential fit.
📡
Pattern Extraction: Extract shared patterns across high-signal candidates (e.g., company lineage, keywords, project scope).
🗂
Archetype Creation: Turn calibration notes into archetype profiles for the hiring team.
⟩_ Example Prompt
⚙ Prompt
"Summarize the common experience patterns among these three storage engineers that indicate readiness for staff-level scope."
📈 Expected Results
✓ Faster calibration loops
✓ Less second-guessing on candidate quality
✓ Increased hiring manager trust
STAGE 05
Outreach Craft
✉️
PRIMARY GOAL
Write high-conversion outreach faster without sounding generic.
🤖 How to use AI
🧪
Tone A/B Testing: Draft multiple message variants and test tone (technical, conversational, value-driven).
🪞
Relevance Mirroring: Mirror language from candidate profiles (project names, tech stacks) for immediate relevance.
📦
Asset Creation: Build a "message bank" for instant reuse across similar roles.
⟩_ Example Prompt
⚙ Prompt
"Write a concise, authentic outreach message to a Staff Virtualization Engineer at Google who's worked on KVM, highlighting impact, ownership, and Crusoe's AI infra mission."
📈 Expected Results
✓ Outreach that feels handcrafted in half the time
✓ Higher reply and interest rates
✓ Consistent voice across the team
STAGE 06
Pipeline Management & Reporting
📊
PRIMARY GOAL
Spend less time formatting updates, more time sourcing.
🤖 How to use AI
📸
Snapshot Creation: Summarize weekly pipeline movement into status snapshots for hiring managers or execs.
🔎
Funnel Analysis: Analyze PTRs (Pass-Through Rates) or stage-drop reasons to spot systemic issues.
⏱
Action Planning: Generate quick follow-up plans for stalled candidates.
⟩_ Example Prompt
⚙ Prompt
"Summarize this pipeline data into a brief update for leadership: outcomes, risks, and next moves."
📈 Expected Results
✓ Instant clarity and consistent reporting
✓ Faster root-cause detection for issues
✓ More time back for high-value sourcing
STAGE 07
Reflection & Continuous Learning
🎓
PRIMARY GOAL
Treat AI as a sourcing partner, not a shortcut.
🤖 How to use AI
🔁
Search Debriefs: Debrief tough searches — "What did I miss?" to uncover gaps.
🔀
Pivot Strategy: Ask for alternative sourcing angles after a failed search cycle.
📖
Technical Fluency: Keep learning technical language through guided Q&A (e.g., "Explain I/O virtualization").
⟩_ Example Prompt
⚙ Prompt
"Suggest 3 alternative sourcing strategies for a search that has stalled for senior-level SDN engineers."
📈 Expected Results
✓ Faster iteration on search strategies
✓ Sharper instincts and domain fluency
✓ Better strategy refresh between cycles

👍 Best Practices
👤
Human Oversight
AI is the engine, you are the pilot. Always verify outputs for accuracy and bias.
🔒
Data Hygiene
Protect candidate privacy. Never input PII into public AI models.
🧪
A/B Testing
Don't settle for the first draft. Test different prompts and outreach tones.
📚
Asset Building
Save successful prompts and messages to a team library for reuse.
📊
Track Impact
Monitor response rates and time-to-calibration to measure ROI.
